# Opzio — PSD/Screenshot to HTML, CSS & JavaScript

A fully responsive animated landing page inspired by the provided Opzio website screenshot.

## Included
- `index.html`
- `styles.css`
- `app.js`
- `assets/` folder with cropped preview assets and SVG marks

## Animations Added
- Preloader animation
- Split-text hero title animation
- Particle canvas background
- Cursor glow effect
- Scroll progress bar
- Scroll reveal animation
- Magnetic buttons
- 3D tilt cards with spotlight hover
- Hero KPI parallax
- Animated price counters
- Interactive cost calculator
- FAQ accordion
- Newsletter feedback
- Back-to-top button
- Responsive mobile menu

## How to Run
Extract the ZIP and open `index.html` in your browser.

For local development, you can also run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
